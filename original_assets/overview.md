# overview

对于一个特殊的化学材料体系：M₁M₂–TiO₂ / Zr‑BTB（MOF）双层复合结构。可调控项包括：TiO₂ 双金属掺杂的金属种类（M₁、M₂）与原子比；以及 Zr‑BTB 上的修饰小分子（需含羧基 –COOH）。注意：MOF 的 linker 固定（BTB），不可调。设计一个 LLM agent 系统自动生成催化剂“配方建议”，并基于已知的“原理类”和“调控性类”文献进行知识整合与推理。还分别知道 7、10 种常见的调控机制（机理、意义等信息）；这些机制更像“影响因素清单”，需要综合考虑，但也可能存在清单之外的因素与相互作用，因此并不完备。
